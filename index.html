JavaScript DOM Cheat Sheet

Selecting Elements

How to select DOM elements using methods like getElementById, querySelector, and querySelectorAll.

// Select by ID
document.getElementById('myElement');

// Select by class
document.querySelector('.myClass');

// Select multiple elements by tag
document.querySelectorAll('div');

Manipulating Elements

Change text content, HTML, attributes, and styles of elements.

// Change text content
element.innerText = 'New Text';

// Change HTML content
element.innerHTML = 'New HTML';

// Change attributes
element.setAttribute('src', 'image.jpg');

// Change style
element.style.color = 'blue';

Event Handling

Respond to user interactions using event listeners. Common event types include click, submit, keydown, and more.

// Click event
element.addEventListener('click', function() {
  alert('Element clicked!');
});

// Form submission event
formElement.addEventListener('submit', function(e) {
  e.preventDefault(); // Prevent default form submission
  console.log('Form submitted');
});

// Keydown event
document.addEventListener('keydown', function(e) {
  console.log('Key pressed: ' + e.key);
});

Common Event Types:

  • click - Triggered by a mouse click
  • mouseover - Triggered when the mouse is over an element
  • keydown / keyup - Triggered when a key is pressed/released
  • submit - Triggered when a form is submitted
  • focus / blur - Triggered when an element gains/loses focus

Traversing the DOM

Navigate the DOM using parentNode, children, nextElementSibling, etc.

// Access parent element
let parent = element.parentNode;

// Access children
let children = element.children;

// Get the next sibling element
let nextSibling = element.nextElementSibling;

// Get the previous sibling element
let prevSibling = element.previousElementSibling;

Creating and Removing Elements

Dynamically create and remove elements from the DOM.

// Create a new element
let newElement = document.createElement('div');
newElement.innerText = 'Hello World';
document.body.appendChild(newElement);

// Remove an element
let element = document.getElementById('myElement');
element.remove();

Event Delegation

Use event delegation to handle events for dynamically created elements by assigning the listener to a parent element.

// Event delegation example
document.querySelector('ul').addEventListener('click', function(event) {
  if (event.target.tagName === 'LI') {
    console.log('List item clicked: ' + event.target.innerText);
  }
});

Working with Forms

Handle form data, input events, and validation using DOM methods.

// Access form input values
let nameInput = document.querySelector('input[name="name"]').value;

// Validate form
if (nameInput === '') {
  console.log('Name is required');
} else {
  console.log('Form is valid');
}
